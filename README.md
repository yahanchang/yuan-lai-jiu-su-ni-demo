# 緣來就塑你

企業內部導師配對平台 Demo，使用 React + Vite + Tailwind CSS 製作。

## 功能

- 響應式 Landing Page
- 註冊、登入與個人資料編輯
- 導師推薦、收藏與導師詳情
- 邀請導師後的交流聊天室
- 社群列表、建立社群、加入社群與發布貼文
- 使用 localStorage 模擬資料保存

## 本機執行

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## GitHub Pages

若要使用 GitHub Pages，執行：

```bash
npm run build:docs
```

推送後，到 GitHub repo 的 Pages 設定中選擇 `Deploy from a branch`，branch 選 `main`，資料夾選 `/docs`。
