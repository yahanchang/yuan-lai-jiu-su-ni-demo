# 緣來就塑你

台塑企業員工共學共創社群平台 Demo，使用 React + Vite + Tailwind CSS 製作。

## 功能

- 響應式 Landing Page
- 註冊、登入與個人資料編輯
- 契合度推薦、收藏與同仁詳情
- 邀請交流後的聊天室
- 社群列表、建立社群、加入社群與發布貼文
- 共學中心、員工課程、Office Hour
- 共創專案與知識成果資料庫
- 使用 localStorage 模擬資料保存

## 本機執行

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## GitHub Pages

若要使用 GitHub Pages，執行：

```bash
npm run build:docs
```

推送後，到 GitHub repo 的 Pages 設定中選擇 `Deploy from a branch`，branch 選 `main`，資料夾選 `/docs`。
